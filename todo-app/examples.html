📚 To-Do List Usage Examples

Getting Started

Open the index.html file in your web browser to start using the To-Do List application. No installation or setup required!

Basic Operations

1️⃣ Adding Your First Task

Step 1: Type your task in the input field at the top

Step 2: Click the "Add Task" button or press Enter

Result: Your task appears in the list with a timestamp

2️⃣ Completing a Task

Method 1: Click the checkbox next to a task

Visual Change: Completed tasks show strikethrough text and appear faded

Statistics Update: The "Completed" counter increases

3️⃣ Editing Tasks

Step 1: Click the "Edit" button on the task

Step 2: A dialog box appears with the current task text

Step 3: Modify the text and click OK

Result: Task is updated instantly and saved

4️⃣ Deleting Tasks

Step 1: Click the "Delete" button on the task

Step 2: Confirm the deletion in the dialog box

Result: Task is removed from the list

Filtering and Organization

Filter by Status

👀 All Tasks

View every task in your list, both completed and active

⏳ Active Tasks

See only tasks that still need to be completed

✅ Completed Tasks

View only tasks you've already finished

🧹 Clear Completed

Remove all completed tasks at once

Statistics Dashboard

Total Tasks: Shows the total number of tasks in your list

Completed: Shows how many tasks you've finished (green)

Remaining: Shows how many tasks are still pending (orange)

These stats update automatically as you add, complete, or delete tasks

Advanced Features

Local Storage & Data Persistence

What is it? Your tasks are saved in your browser's local storage

How it works: Every change is automatically saved

Persistence: Tasks remain even after closing and reopening the browser

Privacy: All data stays on your device - never sent to any server

Export Tasks (Developer Console)

To export your tasks as a JSON file:

app.exportTasks()

This downloads your tasks with a timestamp filename for backup

Import Tasks (Developer Console)

To import previously exported tasks:

app.importTasks(jsonString)

Replace jsonString with your JSON data

Keyboard Shortcuts

Enter Key: While in the input field, press Enter to quickly add a task

Tab Navigation: Use Tab to navigate through buttons

Checkbox: Space to toggle checkbox after tabbing to it

Troubleshooting

Tasks Not Saving?

Solution 1: Check if your browser allows localStorage

Solution 2: Try disabling browser extensions that block storage

Solution 3: Clear browser cache and reload the page

Page Displays But No Tasks Show?

Check: Make sure JavaScript is enabled in your browser

Check Browser Console: Press F12 to see if there are any error messages

Can't Find Exported Tasks?

Location: Check your browser's default Downloads folder

Filename: Look for a file named tasks_[timestamp].json

System Requirements
